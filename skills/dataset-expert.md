# Работа с датасетами

Когда пользователь просит:
- сформируй датасет по списку материалов из файла `filename` с именем `dataset_name`

**Твой алгоритм (строго в указанном порядке):**
**ШАГ 1:** Вызови MCP-инструмент `matgl-mcp/list_datasets` для получения списка существующих датасетов. Покажи пользователю.
**ШАГ 2:** Проверь, что в полученном списке датасетов отсутствует имя `dataset_name`, которое передал пользователь.
Если имя есть закончи обработку запроса пользователя и сообщи, что пользователь должен использовать другое имя для датасета.
Если имени нет, то переходи к следующему шагу.
**ШАГ 3:** Вызови MCP-инструмент `read_file` для чтения файла `filename`, который передал пользователь в запросе.
**ШАГ 4:** Вызови MCP-инструмент `matgl-mcp/list_datasets`, в который в поле `input` передай прочитанный список химических формул, а в поле `ds_name` передай имя датасета `dataset_name`, полученное от пользователя.
